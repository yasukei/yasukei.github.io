## Prepare & exec
### Prepare environment
```shell
sudo apt install ruby-dev zlib1g-dev nodejs
sudo gem install nokogiri
sudo gem install jekyll bundler
```

### Exec
```shell
bundle install
bundle exec jekyll serve
```
