# Headers

```markdown
# Header 1
## Header 2
### Header 3
```

# List

```markdown
- Buleted
- List

1. numbered
2. list
```

# Emphasis

```markdown
**Bold** and _Italic_ and `Code` text
_em tag_
*em tag*
__strong tag__
**strong tag**
```

# Link

```markdown
[LinkText](url) and ![Image](src)
```

# Strikethrough

```markdown
~~striked through~~
```

# Table

```markdown
| Left align | Center align | Right align |
|:-----------|:------------:|------------:|
| aaa        | bbb          | ccc         |
```
